# WildlyPlay — Keyword Research (user-journey driven)

*Framing per Nick: find the keywords so WP's POTENTIAL USERS discover WP. Map = who they are → what they'd search → which WP page must rank. VI/TH-first. Hybrid (keyword in title, analysis in body). $0 method.*

## WP's potential users → their search → target page

### 1. 🎯 Bettor wanting a tip on a SPECIFIC match (highest volume)
- VI: `soi kèo {A} vs {B}`, `nhận định {A} {B}`, `kèo {A} {B}`
- TH: `วิเคราะห์บอล {A} {B}`, `ทีเด็ด {A} {B}`
- EN: `{A} vs {B} prediction`, `{A} vs {B} tips`
- → **WP per-match preview page** (Hybrid: title=keyword, body=transparent analysis + the pick + CLV).

### 2. 🎯 Bettor checking a RESULT (high volume, low intent)
- VI: `kết quả {A} {B}` · TH: `ผลบอล {A} {B}` · EN: `{A} vs {B} result`
- → **WP recap page**, published within minutes of FT (recap-speed + IndexNow). Measure by reach/Discover, not conversion.

### 3. 🎯 Beginner LEARNING betting concepts (informational, E-E-A-T)
- VI: `kèo châu á là gì`, `cách đọc kèo bóng đá`, `CLV là gì`, `value betting là gì`
- TH: `asian handicap คือ`, `วิธีอ่านราคาบอล`
- EN: `asian handicap explained`, `what is closing line value`
- → **WP evergreen pillars** (#1 AH drafted, #2 CLV, #3 Responsible Play) → internal-link DOWN to live picks.

### 4. 🎯 Someone seeking a TRUSTWORTHY / transparent / AI tipster — WP's USP (differentiation lane)
- VI: `tipster uy tín`, `soi kèo chuẩn`, `AI soi kèo`, `dự đoán bóng đá AI`
- TH: `ทีเด็ดบอลแม่นๆ`, `AI ทายผลบอล`
- EN: `transparent football tipster`, `AI football prediction`, `tipster with proven record`
- → **WP About / track-record / Archive pages.** ← This is where WP BEATS competitors: everyone else hides results; WP's public W/L/P + CLV is the answer to exactly this search. Highest-fit, under-served niche.

### 5. 🎯 Someone researching a TEAM / LEAGUE
- VI: `{team} phong độ`, `bảng xếp hạng {league}` · EN: `{league} standings`, `{team} form`
- → **WP standings / matches / team hubs.**

## Opportunity clusters (where to win first — $0, VI/TH-first)
1. **VI/TH per-match long-tail** — huge combined volume, far less competitive than EN. Programmatic = hundreds of fixtures auto. Fastest wins.
2. **"AI soi kèo" / "AI football prediction"** — emerging niche, few competitors, EXACTLY WP's positioning. Own it early.
3. **Transparent-tipster terms** (journey #4) — low competition, perfect USP fit, converts (these searchers WANT what WP uniquely offers).
4. **Informational VI/TH** ("kèo châu á là gì") — low comp, builds E-E-A-T, internal-links to picks.
